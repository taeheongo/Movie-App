# Movie App 2020 ver.

## This application is the fullstack app using Apollo and Graphql.

## Technologies

- HTML
- CSS
- React
- Apollo
- Redux
- REST API
- GraphQL
- Node.js
- Express.js
- AWS(EC2)
- MongoDB(Atlas)
  <br/><br/><br/>

## How to use

<br/>

### start client and server concurrently

<br/>
in file directory.

    npm start

<br/>

### Run server

<br/>

in the ./server

    npm start

<br/>

### Run client

<br/>

in the ./client

    npm start
